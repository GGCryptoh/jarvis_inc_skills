# Official Skills

Official skills are maintained by Jarvis Inc and represent the core capabilities available to all agents in the God View Dashboard.

## Categories

- **Communication** -- Email, Slack, calendar integration via OAuth
- **Research** -- Web search, social media analysis, deep research via LLM
- **Creation** -- Image generation, video generation, document writing, code generation
- **Analysis** -- Data analysis, image analysis, document summarization, translation

## Maintenance

These skills are reviewed and updated by the Jarvis Inc team. To propose changes, submit a Pull Request with a clear description of the change and the rationale behind it.

All Official skill files must pass validation against `schema/skill.schema.json` before merging.
