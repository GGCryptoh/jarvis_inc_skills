# Marketplace Skills

Community-contributed skills for the Jarvis Inc God View Dashboard.

## Contributing

1. Create a folder with your username: `Marketplace/<your-username>/`
2. Add your skill JSON file(s) inside your folder
3. Validate against `schema/skill.schema.json`
4. Submit a Pull Request

See the [root README](../README.md) for full contribution guidelines, schema documentation, and testing instructions.

## Important

Community skills are **not** reviewed or endorsed by Jarvis Inc. Users should review skill definitions carefully before enabling them, paying particular attention to OAuth scopes, API permissions, and external service dependencies.
